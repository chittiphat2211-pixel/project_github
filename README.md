## ตัวอย่าง PHP ง่ายๆ

ไฟล์ตัวอย่างอยู่ที่ `index.php` — หน้าเว็บแสดงข้อความต้อนรับและเวลาปัจจุบัน

การใช้งาน
- เปิดเบราว์เซอร์ไปที่: `http://localhost/project_github/` (หรือ URL ของ Laragon)
- ทดสอบส่งชื่อด้วยพารามิเตอร์: `http://localhost/project_github/?name=สมชาย`

หมายเหตุ
- ต้องรันภายใต้เว็บเซิร์ฟเวอร์ที่รองรับ PHP (Laragon, XAMPP, WAMP ฯลฯ)
